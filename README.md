# PT4Rec
PT4Rec: A Universal Prompt-Tuning Framework for Graph Contrastive Learning-Based Recommendations

## Requirements
```
torchnmf
numba
scipy
torch>=1.7.0
```

## Usage
```
run main.py
(input) PT4Rec
```
